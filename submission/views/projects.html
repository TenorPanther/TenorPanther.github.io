Projects

Candyland

A digital recreation of the classic board game Candyland. Written in C++ and run in the terminal, players had to manage their candies, stamina, and gold in a race to the finish. This recreation was able to support up to 8 players.

Project Turtle

A collaborative project mixing creative writing, visual art, and game design. "Project Turtle" is the codename given to this in-progress video game demo.

The Deiro Mythos

Semi-inspired by DnD and the Cthulhu Mythos, this project started as a short story. To keep up with the expanding world, a website storing the world information and a character "bible" is being created.
